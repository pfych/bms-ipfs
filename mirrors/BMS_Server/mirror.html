Why & What

What is BMS IPFS?

Beatoraja has support for the IPFS protocol. In theory this allows users to play and download charts inside their client. Sadly it is not taken advantage of (This is what I want to change)

Why all this effort?

For new players, getting started with BMS can be difficult. Especially when it comes to downloading charts. Beatoraja has supported IPFS for a long time however it has been up to the table maintainers to add support.

I have created a script to automatically Mirror popular tables, however for now I am just mirroring this small test table.

Due to the way IPFS works, the more people that use it the faster downloads are. I hope, that through offering mirrored tables (which will be slightly behind) I can help more people find out about this cool un-used Beatoraja feature.

If you run or maintain a popular table. Please reach out to me either on discord pfych#0001 or via email anime at tiddi dot es. I am happy to answer any questions and help you take advantage of IPFS

I am currently working to make my script as easy to use as possible. It is open sourced here: https://github.com/pfych/bms-ipfs

How does BMS IPFS work?

IPFS is a peer to peer (p2p) file sharing service. Like torrents. However it allows users to fetch specific files via their unique hash without the need for a torrent file. When a user requests a file their local IPFS node saves the file to their BMS folder but also caches it locally to share with other users on the network (like seeding).

Read more about it here: https://docs.ipfs.io/concepts/what-is-ipfs/

BMS Tables can add an "ipfs" value to their entries. This entry contains an IPFS CID, Beatoraja can use this CID to request the file from the IPFS network. The more people who have indexed or accessed the file the faster this process will be. You can view an example of what this looks like by viewing this tables data.json file here: data.json.

Install steps

Install IPFS Desktop client: https://docs.ipfs.io/install/ipfs-desktop/

Optional steps (Might make IPFS more reliable):

  1. Once IPFS installed and running, search the following hash in the top search bar: 
    QmPjau1s4mjT7t7hCY4aJVJwxwM9Vq87wfFUFLawQj7MXH
    CLICK INSPECT not browse
  2. Wait for the folder contents to appear, This make take a minute.
  3. Once the folder content appears go to the files tab and click import
  4. The click "From IPFS" and enter the same hash: 
    QmPjau1s4mjT7t7hCY4aJVJwxwM9Vq87wfFUFLawQj7MXH
  5. Wait for the folder to import, This may take a few minutes.

Mandatory steps:

  1. Beatoraja has not been patched with custom node support yet, this jar is patched: https://anime.tiddi.es/beatoraja-ipfs.jar
    Beatoraja supports IPFS but not from local nodes. This jar includes a patch to support local nodes
  2. Setup Beatoraja as usual
  3. Add this site as a table
  4. Under the "Other" tab, check "Enable automatic download of BMS by IPFS"
  5. Add your local gateway. This will be in your IPFS app under "Status > Advanced". Its usually: 
    http://127.0.0.1:8080
  6. Launch the game
  7. Go to the "gayme table" table
  8. When you pick a chart you should see in the top corner of your screen download text
  9. It will look like: 
    downloading:ipfs/chart-name-here
  10. Once the file is "found" it will change to look like this (notice size): 
    downloading:ipfs/chart-name-here: [1mb]
    This may take some time.
  11. Your client will automatically refresh your BMS folder
  12. Play the chart! :)
